class Archetype

package awe

Blueprints for fast construction of Entitys.

This can be constructed by using the Archetype.build macro, or by calling the constructor with the composition and component defaults.

Constructor

new (cid:BitVector, defaults:Array<Void ‑> Component>)

Create a new Archetype instance based on its composition and component defaults.

Parameters:

cid

The composition.

defaults

A list of functions that make components.

Methods

create (world:World):Entity

Create a single Entity, using this Archetype as a template.

Parameters:

world

The world to create the entities in.

Returns:

The entity created.

createSome (world:World, count:Int):ArrayList<Entity>

Create multiple entities at once, using this Archetype as a template.

Parameters:

world

The world to create the entities in.

count

How many entities will be created.

Returns:

The list of entities created.
