interface IComponentList<T>

import awe.ComponentList

Represents a list of components.

Variables

The number of entities stored in this list.

Methods

Add the component to this list with the given ID.

Parameters:

id

The Entity to add a component to.

Retrieve the component corresponding associated to the ID.

Parameters:

id

The Entity to retrieve the component for.

Returns:

The component.

get_length ():Int

initialize (world:World):Void

Setup this component list inside the world. This must be ran before other methods are called.

Parameters:

world

The world to initialize this in.

iterator ():Iterator<ComponentListItem<T>>

Iterate through the items in this list.

Returns:

The iterator for this list.

remove (id:Entity):Void

Remove the component corresponding to the ID given.

Parameters:

id

The Entity to remove from this list.
