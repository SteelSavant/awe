class EntitySystem

package awe

extends System

implements SubscriptionListener

import awe.System

Performs operations on entities matching a given Archetype.

Extending this lets you use the '@auto' metadata in front of fields that are a IComponentList<...> or a System to automatically fetch this component list or system from the world in the initialize method.
